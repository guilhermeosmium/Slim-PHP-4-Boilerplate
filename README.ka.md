# ­Ъџђ Slim-PHP-4-Boilerplate

­Ъџђ Boilerplate рЃарЃљрЃЌрЃљ рЃАрЃгрЃарЃљрЃцрЃљрЃЊ рЃЊрЃљрЃўрЃгрЃДрЃЮрЃЌ рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃърЃарЃЮрЃћрЃЦрЃбрЃћрЃЉрЃў **Slim PHP 4-рЃўрЃА** рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ! рЃћрЃА рЃарЃћрЃърЃЮрЃќрЃўрЃбрЃЮрЃарЃўрЃљ рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА рЃЌрЃЦрЃЋрЃћрЃюрЃЌрЃЋрЃўрЃА рЃАрЃарЃБрЃџрЃДрЃЮрЃцрЃўрЃџ рЃАрЃљрЃгрЃДрЃўрЃА рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљрЃА, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃБрЃЎрЃљрЃЋрЃерЃўрЃарЃЊрЃћрЃЉрЃљ рЃАрЃљрЃБрЃЎрЃћрЃЌрЃћрЃАрЃЮ рЃърЃарЃљрЃЦрЃбрЃўрЃЎрЃћрЃЉрЃА MongoDB, PostgreSQL, dotenv-рЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ рЃњрЃљрЃарЃћрЃЏрЃЮрЃћрЃЉрЃўрЃА рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃўрЃАрЃЌрЃЋрЃўрЃА, рЃљрЃАрЃћрЃЋрЃћ рЃФрЃџрЃўрЃћрЃа рЃарЃўрЃњрЃўрЃА рЃЊрЃљ рЃЎрЃћрЃерЃўрЃарЃћрЃЉрЃўрЃА рЃњрЃљрЃЊрЃљрЃгрЃДрЃЮрЃЉрЃўрЃА рЃАрЃЮрЃџрЃБрЃфрЃўрЃћрЃЉрЃА, RabbitMQ рЃЊрЃљ Redis. рЃЊрЃљрЃўрЃгрЃДрЃћрЃЌ рЃЌрЃЦрЃЋрЃћрЃюрЃў PHP рЃърЃарЃЮрЃћрЃЦрЃбрЃћрЃЉрЃў рЃФрЃџрЃўрЃћрЃарЃў рЃЊрЃљ рЃћрЃцрЃћрЃЦрЃбрЃБрЃарЃў рЃЉрЃљрЃќрЃўрЃЌ. Star РГљ рЃЊрЃљ Fork рЃарЃљрЃЌрЃљ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃЮрЃЌ рЃЌрЃЦрЃЋрЃћрЃюрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ рЃърЃарЃЮрЃћрЃЦрЃбрЃерЃў.

## ­ЪЊд рЃърЃарЃћрЃарЃћрЃЎрЃЋрЃўрЃќрЃўрЃбрЃћрЃЉрЃў

- [Docker](https://www.docker.com/get-started)

## ­ЪЏа№ИЈ рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ

1. **рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ .env рЃцрЃљрЃўрЃџрЃў**: рЃЊрЃБрЃЉрЃџрЃўрЃарЃћрЃЉрЃБрЃџрЃћрЃЉрЃљ `.env.example` рЃцрЃљрЃўрЃџрЃў рЃЊрЃљ рЃњрЃљрЃЊрЃљрЃарЃЦрЃЏрЃћрЃЋрЃћрЃЌ `.env`. рЃерЃћрЃЏрЃЊрЃћрЃњ, рЃњрЃљрЃюрЃљрЃљрЃ«рЃџрЃћрЃЌ рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃў рЃарЃЮрЃњрЃЮрЃарЃф рЃАрЃљрЃГрЃўрЃарЃЮрЃљ.

2. **рЃљрЃњрЃћрЃЉрЃљ рЃЊрЃљ рЃАрЃћрЃарЃЋрЃўрЃАрЃћрЃЉрЃўрЃА рЃљрЃдрЃћрЃЉрЃљ Docker-рЃўрЃА рЃЏрЃћрЃерЃЋрЃћрЃЮрЃЉрЃўрЃЌ**:
\```
docker-compose up -d --build
\```

## ­Ъїљ рЃљрЃърЃўрЃА рЃгрЃЋрЃЊрЃЮрЃЏрЃљ
