## 🛠️ קאַנפיגוראַטיאָן

1. **קאַנפיגורירן דעם .env טעקע**: דופּליקאַטע דעם `.env.example` טעקע און נענען צו `.env`. דאַנאַך, דערהייט די וואַריאַבלז ווי דאַרף זיין.

2. **בוי ון און הייב אויף די סערוויסעס מיט Docker**:
\```
