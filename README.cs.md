# 🚀 Slim-PHP-4-Boilerplate

🚀 Boilerplate pro rychlý start projektů s **Slim PHP 4!** Toto repozitář nabízí kompletní výchozí strukturu, integruje nejlepší postupy s MongoDB, PostgreSQL, dotenv pro správu proměnných prostředí a také výkonná řešení pro fronty a mezipaměť, RabbitMQ a Redis. Zahajte své PHP projekty s pevným a efektivním základem. Dejte Star ⭐ a Fork pro použití ve vašem dalším projektu.

## 📦 Předpoklady

- [Docker](https://www.docker.com/get-started)

## 🛠️ Konfigurace

1. **Nakonfigurujte soubor .env**: Duplikujte soubor `.env.example` a přejmenujte na `.env`. Poté aktualizujte proměnné podle potřeby.

2. **Sestavte a spusťte služby pomocí Dockeru**:
\```
docker-compose up -d --build
\```

## 🌐 Přístup k API

Po spuštění služeb pomocí Dockeru můžete přistupovat k API na následující adrese:

[http://localhost:8080/](http://localhost:8080/)
